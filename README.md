# 📊 Customer Churn Analysis

## 📌 Project Overview
This project analyzes customer churn data using Python, SQL, and data visualization techniques. The goal is to identify the factors that influence customer churn and provide insights that can help businesses improve customer retention.
---

## 🚀 Objectives
- Load customer data from a SQLite database.
- Perform data cleaning and preprocessing.
- Explore customer behavior through Exploratory Data Analysis (EDA).
- Identify key factors affecting customer churn.
- Generate meaningful visualizations and business insights.
---
## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- SQLite
- Jupyter Notebook

## 📋 Data Cleaning
The following preprocessing steps were performed:
- Renamed columns for better readability.
- Removed unnecessary columns.
- Converted date columns into datetime format.
- Checked for missing values.
- Verified data types.
- Prepared the data for analysis.

## 📈 Exploratory Data Analysis (EDA)
The project includes:

- Business insights

- Identified patterns among customers who are likely to churn.
